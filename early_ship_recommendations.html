Early Ship Recommendations Guide

Author(s): Samheart564
Current Version: 1.1.4 | Last Updated: November 24th, 2023


Introduction

Here are some good ships players can get extremely early on in Azur Lane!

See the Shop Priority Guide for more detailed shop information.


Starter Ships

You only need to level 1 starter ship; no need to focus on all 3 at the start of the game. They are all fairly equal in strength.


Ship Location Description

Javelin

Map Drop
(all stages except Chapter 1)

Great DD option, especially once fully maxed. Retrofit as soon as possible!

Laffey

Map Drop
(all stages except Chapter 1)

Great DD option, especially once fully maxed. Retrofit as soon as possible!

Z23

Map Drop
(all stages except Chapter 1)

Great DD option, especially once fully maxed. Retrofit as soon as possible!


Guild Shop

See also: Guild Shop (Shop Priority Guide)

Notes:

  • You can refresh the Guild Shop using Guild Coins, DO NOT USE GEMS TO REFRESH SHOPS!

  • Retrofit all ships EXCEPT Helena, whenever possible. Helena Retrofit is lower priority compared to other retrofits listed.

Ship Location Description

Unicorn

Guild Shop - Elite Ship

Unicorn is the best healer in the game. Retrofit as soon as possible!

Helena

Guild Shop - Elite Ship

Helena is a decent damage-dealer early on, and her support capabilities are top-tier for bosses.

Portland

Guild Shop - Rare Ship

Portland is a great tank for mid-late game. Retrofit as soon as possible!

Pennsylvania

Guild Shop - Rare Ship

She is your best easily-accessible BB.


Medal Shop

See also: Medal Shop (Shop Priority Guide)

Notes:

  • The other ships not listed here are mainly for collection purposes, or are sidegrades to cheaper ships listed in this guide.

  • Get one of the below ships from Beginner Rewards, and the other from Medal Shop. Duplicates are fine since they save bulins!

Ship Location Description

Enterprise

Medal Shop - CV

Amazing damage-oriented CV.

Hood

Medal Shop - BB

Great flagship-oriented BC, suitable for any content. Unique Augment is a great improvement over base Hood, but it is not necessary until much later in the game.


Core Data Shop

See also: Core Data Shop (Shop Priority Guide)

Ship Location Description

Independence

Core Data Shop

Post-Retrofit: Amazing support-oriented CV with great damage.

  • Retrofit all ships EXCEPT Yamashiro and Fusou, whenever possible.
