Best General Use for BB Damage (+Hit, +FP)

Type 1 AP Shell

(AKA White Shell)

(Core Data Shop)

Main Gun CRIT DMG +25%

One of the best general use BB auxes in the game, due to its combination of FP and HIT. Usually used alongside Super Heavy Shell, High Standard Fire-Control Radar, Admiralty Fire Control Table.

Best General Use for BB Damage (+FP)

Super Heavy Shell

(AKA Black Shell)

(Core Data Shop)

Main Gun CRIT Rate +8%

One of the best general use BB auxes in the game for damage. Usually used alongside Type 1 AP Shell, High Standard Fire-Control Radar, Admiralty Fire Control Table.

Top Tier CV (post-Retrofit)

Independence

(Core Data Shop)

Has an excellent barrage, plane loadout, efficiencies, and an amazing retrofit skill (Supporting Wings). Requires retrofit to be good. Only high priority if you lack good CVs.
You only need 1 copy of her.

For Guided-missile Destroyers (+DMG)

SY-1 Missile

(Core Data Shop, Joint Battle Mystery Investigation)

Best (and only) Missile equipment in the entire game.

Only usable on Guided-missile Destroyers (DDG).

For Guided-missile Destroyers (+DMG)

Seal of the Four Gods

(Core Data Shop)

If Equipped by an Anshan Class DD: +16% Main Gun DMG, improve All-out-Assault.

Has 50 extra HP over a Repair Toolkit, alongside all the buffs it gives to Anshan Class DDs. Only equippable by DDs!

For Submarines (+OXY)

Improved Snorkel

(Core Data Shop)

Gives a large quantity of OXY to Submarines, allowing them to stay underwater longer (and therefore fire more Torpedoes).

For Submarines (+Geometric Accuracy)

Pressure-Resistant Hull Design

(Core Data Shop)

+20% SPD, moves Submarines forward another 8 units when entering battle.

Moves Submarines closer to enemy ships, making it easier to target and fire Torpedoes at enemies. Some ships' skils move them forward already (ex. U-101), making this equipment redundant.
